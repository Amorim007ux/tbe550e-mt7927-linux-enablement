# Support

## Where to Ask
Use GitHub Issues for:
- reproducible bugs
- hardware compatibility reports
- regression reports after kernel updates

## What to Include
- distro and kernel version
- `lspci -nnk -s 04:00.0`
- `lsusb | grep -i 0489:e116`
- relevant `dmesg -T` excerpts
- steps to reproduce
